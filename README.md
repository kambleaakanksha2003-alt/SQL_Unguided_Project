# SQL_Unguided_Project

---

## 📌 Project Overview

This project focuses on analyzing e-commerce sales data using SQL.
The goal of this project is to explore transactional data and extract meaningful business insights using SQL queries.

The dataset includes information about customers, products, orders, and order details. By performing different SQL operations such as joins, aggregations, and grouping, we can better understand sales performance and customer behavior.

This project demonstrates how SQL can be used for real-world data analysis and decision-making.

---

## 🎯 Objectives

The main objectives of this project are:

* To understand the structure of relational databases
* To perform data analysis using SQL queries
* To analyze sales data and generate business insights
* To practice SQL concepts such as joins, aggregations, and filtering
* To build a portfolio project for data analysis

## 📊 Dataset Description

The dataset used in this project represents a simplified e-commerce system.
It contains four main tables:

### 1️⃣ Customers

Contains information about customers who made purchases.

Columns:

* customer_id
* customer_name
* city
* signup_date

### 2️⃣ Products

Contains details about products available in the store.

Columns:

* product_id
* product_name
* category
* price

### 3️⃣ Orders

Stores order-level information.

Columns:

* order_id
* customer_id
* order_date

### 4️⃣ Order Details

Contains detailed information about each product within an order.

Columns:

* order_detail_id
* order_id
* product_id
* quantity

---

## 🔍 SQL Analysis Performed

### 1️⃣ Total Revenue Calculation

Calculated the total revenue generated from all orders by multiplying product price with quantity sold.

### 2️⃣ Top Selling Products

Identified products with the highest sales volume based on quantity sold.

### 3️⃣ Revenue by Product Category

Analyzed which product categories generated the highest revenue.

### 4️⃣ Top Customers

Identified customers who contributed the most revenue to the business.

---

## 📈 Key Insights

Some insights derived from this analysis include:

* Certain product categories contribute significantly to overall revenue.
* A few products generate the majority of total sales.
* Repeat customers play an important role in overall revenue generation.
* Customer purchase patterns can help businesses improve marketing strategies.
